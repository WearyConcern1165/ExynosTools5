===========================================
INSTRUCCIONES DE INSTALACIÓN
===========================================

1. DESCARGAR:
   - Archivo: xclipse_tools_winlator_ready.tar.zst
   - Tamaño: ~50KB
   - Formato: tar.zst (comprimido)

2. INSTALAR EN WINLATOR:
   ```bash
   # Copiar el archivo a Winlator
   cp xclipse_tools_winlator_ready.tar.zst /path/to/winlator/
   
   # Extraer el paquete
   cd /path/to/winlator/
   tar -xzf xclipse_tools_winlator_ready.tar.zst
   ```

3. VERIFICAR INSTALACIÓN:
   ```bash
   # Verificar que el archivo existe
   ls -la /usr/lib/libxeno_wrapper.so
   
   # Verificar que no está vacío
   file /usr/lib/libxeno_wrapper.so
   ```

4. CONFIGURAR:
   Crear archivo: /data/data/com.winlator/files/xclipse_tools.conf
   
   Contenido:
   ```
   # Xclipse Tools Configuration
   performance_mode=2
   shader_cache_enabled=1
   bc4_emulation_enabled=1
   debug_logging=1
   advanced_features_enabled=1
   memory_optimization=2
   gpu_scaling=2
   fps_monitoring=1
   thermal_protection=1
   dxvk_compatibility=1
   dynamic_rendering_emulation=1
   ```

MODOS DE RENDIMIENTO
===========================================

1. HIGH (performance_mode=1):
   - Máximo rendimiento
   - Mayor consumo de batería
   - Present mode inmediato

2. BALANCED (performance_mode=2):
   - Equilibrio rendimiento/batería
   - Modo recomendado

3. POWER_SAVE (performance_mode=3):
   - Ahorro de batería
   - Rendimiento reducido

===========================================
SOLUCIÓN DE PROBLEMAS
===========================================

PROBLEMA: "no vendor ICD found"
SOLUCIÓN: Asegúrate de tener un driver Vulkan instalado

PROBLEMA: Bajo rendimiento
SOLUCIÓN: Cambia performance_mode=1 en la configuración

PROBLEMA: Alto uso de memoria
SOLUCIÓN: Cambia memory_optimization=3

PROBLEMA: Juegos no inician
SOLUCIÓN: Verifica que bc4_emulation_enabled=1
